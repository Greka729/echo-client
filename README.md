# echo-client

server.py

Запускается на 127.0.0.3:65432

Принимает входящие подключения

Принимает данные от клиента

Выводит полученные данные в консоль

Автоматически закрывает соединения

client.py

Подключается к серверу по адресу 127.0.0.3:65432

Отправляет сообщение "Hello world"

Принимает ответ от сервера

Выводит полученный ответ в консоль

# Настройка
Для изменения адреса или порта подключения отредактируйте константы в обоих файлах:

HOST = "127.0.0.3"  # IP-адрес
PORT = 65432        # Порт
